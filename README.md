## About me

I'm Nishant Makwana, a Full-Stack Software Engineer who turns complex engineering problems into fast, reliable, and scalable products — the kind that handles real load and real users. I program in C++ and Python, build interfaces with React, TypeScript, and Tailwind CSS, and ship backends on Node.js with Express.js. My systems baseline is Linux (Ubuntu); data lives in PostgreSQL, MongoDB, and Redis. For analytics and modeling I use NumPy and Pandas, and for machine learning scikit-learn and PyTorch. I run workloads on AWS, package services with Docker, front them with Nginx, and ship changes with Git.

In production, I've shipped a hospital-ops platform serving 3 hospitals, an IoT patient-vitals pipeline ingesting 500+ metrics/hour, and a real-time collaborative code editor scaling to 100+ concurrent users at sub-200ms latency — with 99.9% uptime, zero regression defects, and 40% bundle reduction to show for it. Outside engineering, I founded a Coding Club of 150+ students, led the team that shipped a college ERP portal for 2,100+ users, and won 3 national hackathons. Currently finishing my B.E. in IT at GTU (CGPA 8.0/10, May 2026) — actively seeking SDE roles at high-impact product companies.
